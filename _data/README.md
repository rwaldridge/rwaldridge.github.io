# Sample Data Files for Academic Portfolio

This directory contains example YAML files that demonstrate the data structure for your academic portfolio site.

## Files

- `publications.yml` - Example bibliography entries
- `repos.yml` - Example GitHub repository highlights
- `engagement.yml` - Example speaking, media, and advisory entries
- `projects.yml` - Example research project metadata

## Usage

Copy these files to your Jekyll site's `_data/` folder and customize with your own content.
